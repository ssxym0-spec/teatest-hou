# 变更日志: 制作批次列表与按钮修复
**日期**: 2025-11-27 00:00
**任务**: 修复制作批次名称显示 undefined 以及按钮未绑定 ID 的问题

## 📂 文件变更
### 修改 (Modified)
- `views/traceability-management.html`:
  - 在 `renderBatchList` 中，批次名称统一从 `batch.batch_number || batch.batchNumber` 读取，避免显示 `undefined`。
  - 兼容 `production_date` / `productionDate` 字段，日期缺失时显示“未设置”。
  - “批次编辑”与“制作编辑”按钮的 `onclick` 事件改为使用 `batch.id || batch._id` 作为参数，避免发起 `/api/batches/undefined` 请求。

### 新增 (Created)
- `docs/changelogs/0044_制作批次列表与按钮修复.md`: 记录本次制作批次前端修复。

### 删除 (Deleted)
- 无

## 💡 技术说明 (Technical Notes)
- 通过在前端做轻量级字段兼容，既支持新 Prisma 返回的驼峰字段，也兼容旧 Mongo 风格字段，减少后续迁移摩擦。
