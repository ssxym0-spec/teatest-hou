# 变更日志: 修复静态资源根路径
**日期**: 2025-11-27 16:25
**任务**: 修复人员头像静态资源无法加载的问题

## 📂 文件变更
### 修改 (Modified)
- `src/server.ts`: 将 `/uploads` 静态目录改为 `express.static(path.join(process.cwd(), 'public/uploads'))`，避免 `__dirname` 指向 `src` 目录导致路径错误。

### 新增 (Created)
- `docs/changelogs/0018_修复静态资源根路径.md`: 记录此次静态资源路径修复。

### 删除 (Deleted)
- 无

## 💡 技术说明 (Technical Notes)
- 通过使用 `process.cwd()` 锁定项目根目录，确保无论在 ts-node 还是编译输出环境下，`/uploads` 路由都能正确映射到 `public/uploads`。
